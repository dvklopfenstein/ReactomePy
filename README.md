## Scripts

### 1) Pathway Scripts
Pathway scripts use Reactome data saved into Python modules.    
Neo4j does not need to be running to use these scripts.    

### 2) Neo4j Scripts
Neo4j scripts require that Neo4j loaded with Reactome is working on your machine.
